# gomoku-ai-spec

五子棋双人对战单页应用（Vue 2.7 + Pinia，JavaScript）。

## 功能
- 15×15 棋盘，允许长连
- 双人轮流落子、胜负弹窗提示
- 当前回合显示、重新开始
- Canvas 拟物风格渲染

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```

## 手工测试

1. 打开页面后直接落子，确认黑子先手。
2. 连续落子形成横/竖/斜连五，出现胜负弹窗。
3. 胜负弹窗出现后尝试继续落子，确认棋盘被锁定。
4. 点击“重新开始”，确认棋盘清空且回合回到黑子。
5. 调整窗口至 1280×720 左右，确认布局仍可用且棋盘未溢出。
