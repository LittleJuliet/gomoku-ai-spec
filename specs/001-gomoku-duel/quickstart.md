# Quickstart: 五子棋对战

## Prerequisites

- Node.js 18+
- npm 或 pnpm

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tests

```bash
# unit tests
npm run test:unit

# e2e tests
npm run test:e2e
```

## Notes

- 若脚本名称不同，请按实际脚本调整。
- 应用为本地双人对战，不依赖后端服务。
