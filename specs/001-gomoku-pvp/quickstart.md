# Quickstart: 五子棋双人对战

**Date**: 2026-01-19

## Prerequisites

- Node.js 18+（或项目已约定版本）
- npm（或项目统一的包管理器）

## Install

```bash
npm install
```

## Run (Dev)

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Notes

- 代码必须使用 JavaScript，禁止引入 TypeScript
- 本项目为本地双人对战，不包含联网功能
