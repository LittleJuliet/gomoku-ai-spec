# UI棋盘模块

## 目的
负责棋盘与棋子的 Canvas 渲染，并处理落子交互。

## 模块概述
- **职责:** 绘制棋盘、处理点击、调用状态更新、触发重绘
- **状态:** ✅稳定
- **最后更新:** 2026-01-19

## 规范

### 需求: 棋盘渲染与交互
**模块:** ui-board
使用 Canvas 绘制 15×15 棋盘与棋子，响应点击落子，游戏结束后禁止继续落子。\n+实现位置: `src/components/BoardCanvas.vue`

#### 场景: 玩家落子
前置条件: 当前回合玩家点击棋盘空格
- 落子成功并更新回合
- 若触发胜利则弹窗提示

### 需求: 当前回合显示
**模块:** ui-board
在主界面显示当前回合玩家（黑/白）。

#### 场景: 回合切换
前置条件: 玩家成功落子
- 回合提示切换为另一方

## API接口
无（组件内通过 Pinia 读取状态与提交落子动作）

## 数据模型
无
